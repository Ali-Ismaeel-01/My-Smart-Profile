Fun Facts About Web Development

  • The first web browser was called WorldWideWeb and was created by Tim Berners-Lee in 1990.
  • CSS (Cascading Style Sheets) was first proposed by Håkon Wium Lie in 1994.
  • The first version of JavaScript was created in just 10 days by Brendan Eich in 1995.
  • The HTML5 specification was finalized and published in 2014, but developers started using it years before.
  • Bootstrap was developed by Mark Otto and Jacob Thornton at Twitter in 2011.
